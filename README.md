# 🌍 Azure Language Translator

A modern and responsive **Language Translation Web Application** built using **HTML, CSS, and JavaScript**, powered by the **Microsoft Azure Translator API**.

This project provides a professional multi-step user interface where users first enter their Azure Subscription Key and Region, then translate text instantly into multiple languages through a sleek and interactive dashboard.
